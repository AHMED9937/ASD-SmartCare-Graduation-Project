DIR (Fixed Class)
•	Attributes:
o	height (double): Height of the screen.
o	width (double): Width of the screen.
o	colors (List<String>): Optional list of colors.

•	Methods:
o	textDescription(fontSize: double, color: String): Returns a text description.
o	textHeader(fontSize: double, color: String): Returns a text header.
o	button(height: double, width: double, color: String): Returns button information (can handle multiple buttons).
o	image(link: String, height: double, width: double, property: String): Returns an image with properties.

Onboarding Screen in Onboarding Dir
•	Attributes:
o	title (String): Title of the onboarding screen.
o	description (String): Description of the onboarding screen.
o	imageLink (String): Link to the image.
•	Methods:
o	Constructor: Requires title, description, and imageLink. These must not be null or empty.


________________________________________
Onboarding Navigation Screen
•	Attributes:
o	screens (List<OnboardingScreen>): List of onboarding screens.
o	currentIndex (int): Tracks the current screen index (default is 0).
•	Methods:
o	addScreen(screen: OnboardingScreen): Adds an onboarding screen to the list.
o	getScreens(): Returns all onboarding screens.
o	displayScreens(): Displays details of the current screen, along with:
	Slider: Indicates which screen is currently displayed (e.g., dots or numbers).
	Skip Button: Allows skipping to the last screen.
	Next Button: Moves to the next screen or finishes if on the last screen.
o	nextScreen(): Advances to the next screen.
o	skipToLastScreen(): Skips directly to the last screen
